Principais formatações

Negrito / Destaque

Nessa frase o termo POKEMON está em negrito, pois eu usei a tag <b> (tag não semântica)

Já nessa frase o termo DESTAQUE é uma palavra forte/em destaque, por isso usei a tag <strong>(tag semântica, tem significado)

Itálico / Enfâse

Nesta frase a palavra ITÁLICO está em itálico, usei a tag <i> (tag não semântica)

Já nesta frase a palavra EMPHASIS está em enfâse (tag semântica, tem significado)

PREFERÊNCIA USAR NO HTML AS TAGS <strong> e <em> AO INVÉS DE <b> e <i>

CRIANDO UM TEXTO PEQUENO

Nesta frase o termo basquete está pequeno (usei a tag <small>)

TEXTO DELETADO

Podemos criar um texto excluído (texto que deve ser lido mas não considerado) usando a tag <del>

TEXTO INSERIDO

Podemos criar um texto inserido (texto que foi inserido) usando a tag <ins>
